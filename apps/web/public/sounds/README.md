# Sound Assets

This directory contains audio files for the Motion Layer animations.

## Required Files

| File | Trigger | Duration |
|------|---------|----------|
| `chip_slide.mp3` | Bet animation start | ~0.3s |
| `chip_impact.mp3` | Chip lands in pot | ~0.2s |
| `card_slide.mp3` | Card deals to player | ~0.3s |
| `card_flip.mp3` | Card reveals in showdown | ~0.2s |
| `win_fanfare.mp3` | Pot awarded to winner | ~1.0s |
| `fold.mp3` | Player folds cards | ~0.2s |
| `check.mp3` | Player checks | ~0.1s |
| `timer_tick.mp3` | Action timer warning | ~0.1s |

## Notes

- Sounds are triggered at animation END (impact) for "heaviness" feel
- Use short, crisp sounds (< 1s for most)
- Recommend royalty-free sources:
  - [Freesound.org](https://freesound.org)
  - [Zapsplat](https://www.zapsplat.com)
  - [Mixkit](https://mixkit.co/free-sound-effects/)
